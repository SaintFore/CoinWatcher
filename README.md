# 🪙 CoinWatcher

## 🚀 简介

CoinWatcher 是一个使用 Streamlit 构建的加密货币价格监控应用。它可以让你实时跟踪多种加密货币的价格，并计算你的投资盈亏。

## ✨ 主要功能

- **多种货币支持**: 支持 `Bitcoin`, `Ethereum`, `Dogecoin` 等多种主流加密货币。
- **多种法定货币**: 支持以 `美元 (USD)` 和 `人民币 (CNY)` 计价。
- **历史价格图表**: 以图表形式展示加密货币的历史价格走势。
- **实时价格**: 显示当前最新的加密货币价格。
- **盈亏计算器**: 根据你的买入成本和持仓数量，计算你的投资盈亏。

## 🛠️ 如何使用

1.  **安装依赖**:
    ```bash
    pip install -r requirements.txt
    ```
2.  **运行应用**:
    ```bash
    streamlit run app.py
    ```
3.  **登录**:
    在侧边栏输入你的名称并点击登录。
4.  **选择货币**:
    在侧边栏选择你想要监控的加密货币和计价的法定货币。
5.  **查看价格**:
    主界面会显示加密货币的历史价格图表和当前价格。
6.  **计算盈亏**:
    在下方的盈亏计算器中输入你的买入成本和持仓数量，即可计算你的投资盈-亏。
